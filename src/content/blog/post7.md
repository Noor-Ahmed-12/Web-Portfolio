During my internship with **Interns Pakistan**, I developed this React-based fitness hub. The project focuses on "Seamless Content Delivery"—integrating multiple data streams to provide a holistic user experience.

### Technical Engineering
* **API Orchestration:** Leveraged **RapidAPI** to fetch a dataset of 1,000+ exercises and synchronized it with the **YouTube Search API** for real-time instructional videos.
* **State Management:** Utilized React Hooks (`useState`, `useEffect`) to handle complex filtering by muscle group and equipment.
* **UI/UX:** Designed with **Material-UI (MUI)** to ensure a mobile-first, responsive experience featuring interactive carousels.



### Knowledge Sharing
I documented the development process on my YouTube channel, reaching **7,000+ developers**. The tutorial covers API key security, custom hooks, and deploying React apps to the **Netlify** cloud platform.
